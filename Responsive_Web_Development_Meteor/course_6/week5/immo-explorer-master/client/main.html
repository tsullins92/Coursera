About

Immo-explorer help users find out the right price for their property, and prospective users check if a price for a property is right

It has 3 sections (a search form, a map and an "about" section)

How to run:

  • meteor npm install --save @babel/runtime
  • meteor run

Limitations:

  • The app works for Paris only, because the database with all France would have been really huge

Bugs:

  • no input controls in the search fields, client-side (on server side, any negative input for surface or # rooms is ignored)
  • map controls are not implemented yet
  • map markers are basic, they'll be replaced by an icon displaying the price of the property
  • accounts not implemented yet (no possibility to create a list of searches per user)
  • probably many others
